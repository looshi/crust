# Crust

The crustiest web synth.

## Install and Run

```sh
# Clone repo and install dependencies
git clone git@github.com:looshi/crust.git
cd crust
npm i
```

```sh
# Run in localhost
npm start
```

```sh
# Build assets for deployed site, will appear in dist folder
npm run build-prod
```

```sh
# run tests
npm run test
```
